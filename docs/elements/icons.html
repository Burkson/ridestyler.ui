Icons#

ridestyler.ui includes its own custom set of icons that you can use in your interface.

<i class="icon icon-arrow-right"></i>

Add a icon class and the correct icon-[name] class to an element to display an icon using a pseudo :before element.

Font Awesome#

All of the icons from have been included. Use their website or browse the /src/icons/font-awesome-solid folder to discover icon names.

Custom icons#

Custom icons can easily be added in the future by adding them to the src/icons folder and rebuilding the fonts.

Icon sizes#

Use icon-2x, icon-3x and icon-4x classes to increase icon sizes. You can set font-size to have different icon sizes.

icon-2x (2rem)

icon-3x (3rem)

icon-4x (4rem)

<i class="icon icon-2x icon-camera-retro"></i>

SCSS Enhancements#

A variable containing the unicode value for each icon is included and can be used in place of the .icon classes. To use the custom variables extend the %icon placeholder selector (or add the font-family: 'rsui-icons'; to your class), and use the variable in the :before or :after pseudo-class. 

.my-custom-icon {
  @extend %icon;

  &:before {
    content: $icon-code-camera-retro;
  }
}
